
3. **Set Up Environment Variables:**

    - Get your gemini API key here : https://ai.google.dev/gemini-api/docs/api-key
    - Add your API key to the `.env` file:
      ```
      apikey=YOUR_API_KEY_HERE
      ```
